# Awesome Codex Skin

**English** | [中文](README.md)

> A curated directory of skins, themes, theme-creation tools, and runtime projects for OpenAI Codex Desktop.

This directory lists third-party open-source projects. Inclusion does not imply OpenAI endorsement or a security guarantee. Before changing a theme, read each project's installation, rollback, and compatibility guidance. Tools that use CDP or debugging ports should be used only in a trusted local environment.

## Theme management and skin-switching tools

- [HeiGe Codex Skin Studio](https://github.com/HeiGeAi/heige-codex-skin-studio) — A macOS / Windows theme studio with presets, image color extraction, a theme center, and restoration.
- [Codex Styler](https://github.com/xuhuanstudio/codex-styler) — An open-source visual theme editor, theme library, and 2D companion tool.
- [Codex AutoSkin](https://github.com/Finderchangchang/codex-autoskin) — An agent-native tool that generates a Codex Desktop skin from an image.
- [CodeDrobe Desktop](https://github.com/CodeDrobe/desktop) — A Codex / WorkBuddy desktop theme manager and theme store.
- [CodexNextTheme](https://github.com/pikapikaspeedup/CodexNextTheme) — A macOS tool for one-click skin switching, fine tuning, desktop companions, and a theme store.
- [Codex-NN](https://github.com/slovx2/Codex-NN) — A cross-platform visual theme manager for Codex Desktop.
- [codex-skin-switcher](https://github.com/bytefer/codex-skin-switcher) — A free and open-source cross-platform theme and skin switcher.
- [codex-skin](https://github.com/charmber/codex-skin) — A macOS skinning tool with backgrounds, color schemes, and themes.

## Creators and automation

- [codex-theme-creator](https://github.com/swording-k/codex-theme-creator) — Creates, installs, and validates a macOS Codex Desktop theme from a sentence or reference image.
- [codex-autoskin](https://github.com/hsir2411-dotcom/codex-autoskin) — Generates a custom skin from an image without modifying core application files.
- [codex-theme-x](https://github.com/focuxdot/codex-theme-x) — Converts an image into a Codex theme.
- [Codex Dream Skin Studio](https://github.com/huoxuhaohaode/Codex-Dream-Skin-Studio) — A visual tool for creating, switching, and sharing Codex Desktop themes.
- [SkinDex / codex-qq-skin](https://github.com/angziii/codex-qq-skin) — Declarative skin-package validation, installation, and local loopback-CDP runtime tooling.

## Inclusion criteria

Pull requests are welcome. To keep the directory useful, projects should:

1. Clearly serve OpenAI Codex Desktop theming, skinning, or a directly related workflow;
2. Have a public repository, clear installation and rollback guidance, and a license;
3. State whether they modify official application files, use CDP, and which platforms they support;
4. Provide a stable in-repository path and attribution when including a real running screenshot. Real screenshots are preferred.
